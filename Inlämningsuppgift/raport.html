Fr att verkligen testa om detta stmmer har jag ven valt att gra ett hypotestest (t-test) fr att verkligen kontrollera att det r ngon skillnad p populationerna.

Fr att gra t-testet brjade jag med att stta upp hypotesen fr den, jag vljer ven att gra testet med en signifikant niv p 95%.

$H_A: \mu_{usa} \neq \mu_{eu}$

$H_0: \mu_{usa} = \mu_{eu}$

$a = 0.05$

Innan jag gr t-testet gr jag ett f-test fr att ta reda p om jag ska stta equal_var till True or False.

Jag rknar ut f-testet igenom att anvnda numpy fr att rkna ut f-stat och scipy fr att rkna ut f-crit.

from scipy.stats import f

# kalkylera F-stat igenom att ta var av usa och eu cars och dela dem p varandra.
f_stat = np.var(df_cars[df_cars["origin"] == "usa"]["horsepower"], ddof=1) / np.var(
    df_cars[df_cars["origin"] == "europe"]["horsepower"], ddof=1
)

# kalkylera df1 och df1 igenom n - 1.
df1 = len(df_cars[df_cars["origin"] == "usa"]) - 1
df2 = len(df_cars[df_cars["origin"] == "europe"]) - 1

# alpha value (significance).
a = 0.05

# kalkylera f_crit.
f_crit = f.ppf(q=a, dfn=df1, dfd=df2)

# printar slutsats.
print(f"f-crit: {round(f_crit, 2)}, f-stat: {round(f_stat, 2)}, f-crit > f-stat = {f_crit > f_stat}")

f-crit: 0.74, f-stat: 3.92, f-crit > f-stat = False

Som man kan se frn svaret p f-testet r f-crit inte strre n f-stat vilket innebr att de tv varianserna p det tv populationerna kan ses som lika, vilket betyder att equal_var = True.

Efter att ha tagit reda p vad equal_var ska vara gick jag vidare till att gra t-testet, fr att gra t-testet anvnde jag ttest_ind funktionen, dr jag satte a som hstkrafter fr USA och b fr EU.

from scipy.stats import ttest_ind

result = ttest_ind(
    a=df_cars[df_cars["origin"] == "usa"]["horsepower"],
    b=df_cars[df_cars["origin"] == "europe"]["horsepower"],
    equal_var=True,
    alternative="two-sided",
)
print(result)

Ttest_indResult(statistic=7.681404142807087, pvalue=2.0653368475960042e-13)

Som man kan se frn resultatet frn t-testet r p-value mycket mindre n det givna signifikant vrdet vilket innebr att man kan kasta noll hypotesen, efter som att p-value r s litet betyder det ven att det r en stor skillnad p medelvrdet av de tv populationerna.

Detta betyder ven att det fr att svaret var mer eller mindre korrekt. detta ledder ven till att svaret p den frsta frgestllningen r ja det r en skillnad p hstkrafter mellan amerikanska och europeiska.
